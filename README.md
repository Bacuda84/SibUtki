## Описание

Приложение состоит из двух экранов:
1. **Основной экран** - экран, где можно выбрать день и посмотреть расписание
2. **Модальный экран** - появляется когда нажимаете на кнопку группы. Просто экран с выбором группы из предложенных институтов и курсов

## Технологический стек

- **UIKIt** - для создания UI
- **MVP** - архитектурный паттерн
- **UserDefaults** - для хранения данных пользователя
- **URLSession** - для работы с API


## Тестирование

Проект покрыт unit-тестами для проверки:
- Валидности сериализации и дисериализации json
- Сохранения и загрузки данных пользователя
- Работы с сетевыми запросами

## Что нового дал мне проект 
- До проекта читал книгу Modern Auto Layout и около 90% книги мне как раз понадобилось для написания внешнего вида. 
- Также работал с кастомными ячейками, прошупал некоторые подводные камни в работе с таблицей, в частности сохранения данных ячейки при resiz'инге, когда в моменте ячейка отгружается из памяти и создаётся заново.
- Также работал немного с кастомной UIRefreshControll, на видео видно, как утка крутится при обновлении таблицы. 
- Поработал с жестами для быстрой смены дня/недели, что в прочем, удобно.
